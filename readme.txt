A simple personal financial bookkeeping system.

== Description ==

A simple financial bookkeeping journal for keeping track of personal expenses and income.
This plugin gives you an extra top-level menu item in the WordPress administration back-end,
under which you can enter payment and receipt data, view monthly summaries, and generate invoices.
